TECH STACK
1.Frontend: React.js, Bootstrap  
2.Backend: Node.js, Express  
3.Database: MongoDB  
4.Auth: JWT, bcrypt  

SETUP
1. Clone the repo
2. Install dependencies in frontend and backend
3. Add required environment variables
4. Start both servers

RUNNING THE APP
Start backend and frontend using:
npm start
